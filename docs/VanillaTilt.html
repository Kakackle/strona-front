extendSettings()

Method return patched settings of instance

Parameters:
Name Type Description
settings.reverse boolean

reverse the tilt direction
settings.max number

max tilt rotation (degrees)
settings.startX startX

the starting tilt on the X axis, in degrees. Default: 0
settings.startY startY

the starting tilt on the Y axis, in degrees. Default: 0
settings.perspective number

Transform perspective, the lower the more extreme the tilt gets
settings.easing string

Easing on enter/exit
settings.scale number

2 = 200%, 1.5 = 150%, etc..
settings.speed number

Speed of the enter/exit transition
settings.transition boolean

Set a transition on enter/exit
settings.axis string | null

What axis should be enabled. Can be "x" or "y"
settings.glare boolean

if it should have a "glare" effect
settings.max-glare number

the maximum "glare" opacity (1 = 100%, 0.5 = 50%)
settings.glare-prerender boolean

false = VanillaTilt creates the glare elements for you, otherwise
settings.full-page-listening boolean

If true, parallax effect will listen to mouse move events on the whole document, not only the selected element
settings.mouse-event-element string | object

String selector or link to HTML-element what will be listen mouse events
settings.reset boolean

false = If the tilt effect has to be reset on exit
settings.reset-to-start boolean

true = On reset event (mouse leave) will return to initial start angle (if startX or startY is set)
settings.gyroscope gyroscope

Enable tilting by deviceorientation events
settings.gyroscopeSensitivity gyroscopeSensitivity

Between 0 and 1 - The angle at which max tilt position is reached. 1 = 90deg, 0.5 = 45deg, etc..
settings.gyroscopeSamples gyroscopeSamples

How many gyroscope moves to decide the starting position.
